How to test SpringBoot Apps?:
    Annotation 	            Purpose	                                                Level
    @WebMvcTest	            Tests only the controller; mocks everything else.	    Unit
    @DataJpaTest	        Tests only the repository/database layer.	            Data
    @RestClientTest	        Tests clients that call external APIs.	                Client
    @SpringBootTest	        Loads the full context; can start a real server.	    Integration
